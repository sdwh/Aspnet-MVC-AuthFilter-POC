# Aspnet-MVC-AuthFilter-POC

示範如何藉由客製 Authentication Filter, Authorize Attribute 來實現 ASP.NET MVC 5 的驗證與授權。

## 使用步驟

1. 下載
2. 重新 Build
3. 清除專案
4. 重建專案
5. 確認 web.config ConnetionString 的 source
6. Nuget Manager > Update Database
